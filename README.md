# DropTimer.gg

A modern web app for tracking countdowns to upcoming video game releases and drops across all major platforms.

## Tech Stack

- **Astro** - Modern web framework
- **TailwindCSS** - Utility-first CSS
- **TypeScript** - Type safety
- **Astro View Transitions** - Smooth page navigation

## Getting Started

### Install Dependencies

```bash
npm install
```

### Development

```bash
npm run dev
```

Visit `http://localhost:4321` to see your app.

### Build

```bash
npm run build
```

### Preview

```bash
npm run preview
```
